# POS Admin Website

This is the admin website for the Restaurant POS system, built with Next.js, React, and Tailwind CSS.

## Prerequisites

Make sure you have the following installed:
- [Node.js](https://nodejs.org/) (Version 18.x or above recommended)
- [pnpm](https://pnpm.io/) (Package manager)

## Setup and Installation

1. Navigate to the project directory:
   ```bash
   cd pos-admin-website
   ```

2. Install the dependencies using pnpm:
   ```bash
   pnpm install
   ```

## Running the Website Locally

To start the development server:

```bash
pnpm dev
```

This will run the app in development mode. Open [http://localhost:3000](http://localhost:3000) in your browser to view the website. The page will reload when you make changes.

## Available Scripts

In the project directory, you can run the following commands:

- **`pnpm dev`**: Starts the development server.
- **`pnpm build`**: Builds the app for production to the `.next` folder. It correctly bundles React in production mode and optimizes the build for the best performance.
- **`pnpm start`**: Starts the production server. You must run `pnpm build` before running this command.
- **`pnpm lint`**: Runs ESLint to check for linting errors in the codebase.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **UI Library**: [React](https://react.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
